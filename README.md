# Julis-WebOS

A personal web-based operating system built with HTML, CSS, and JavaScript.

## About

Julis-WebOS is a desktop-inspired website that recreates the experience of a small operating system inside the browser. The project combines draggable windows, desktop applications, interactive content, and a custom visual design to create a unique personal web environment.

The goal of the project is to explore front-end development while building something creative, interactive, and personal.

---

## Features

### Desktop Environment

* Custom desktop wallpaper
* Live clock and date display
* Desktop application icons
* Window-based interface
* Draggable windows
* Open and close window functionality

### Welcome Window

* Personal introduction
* Interests and goals
* Social links
* Custom profile design

### Friends Window

* Dedicated image gallery window
* Draggable and closable

### Doodles Window

* Displays personal artwork and sketches
* Draggable and closable

### Locked In Window

* Additional themed content window
* Integrated into the window management system

### Star Journal App

A custom journaling and review application.

Features:

* Write journal entries
* Review books, movies, games, and more
* 1–5 star rating system
* Save reviews directly inside the application
* Interactive user interface

### Constellation Explorer App

A space-themed educational application.

Features:

* Browse different constellations
* Learn interesting facts about each constellation
* View constellation images
* Favorite constellation system
* Interactive constellation selection menu

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

No external frameworks were used.

## Challenges

During development, several technical challenges were solved:

* Building a reusable window system
* Creating draggable windows
* Preventing dragging from interfering with text inputs
* Implementing interactive app functionality
* Managing multiple independent application windows
* Creating dynamic content updates for the Constellation Explorer
